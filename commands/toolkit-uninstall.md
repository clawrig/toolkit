# Toolkit Uninstall

Run the interactive uninstall script.

## Steps

1. Run `python3 ${CLAUDE_PLUGIN_ROOT}/scripts/uninstall.py`
2. The script will show installed tools and prompt which to remove
3. Display the results to the user
